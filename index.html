  • Python Engine

    I need a simple python engine that will let me to run different python applications. A simple setup will do fine. Later I can implement the Command pattern and then the MVC pattern. To simplify the integration of the engine in new applications.

  • Terminal Enviroment

    I will have a terminal Input and terminal Output functionalities. The engine has to be able to take input from the terminal Input and pass it to the execute functionality. The execute functionality needs to execute the internal logic.

    • Terminal Enviroment \ Functional Stage

      For the start pure functional programing will be enoght. Once the main engine functionalities are layed down. The engine can proceed to the object oriented programing stage.

      • Loop Functiolaity

        I need a untimed terminable engine loop.

      • Input Functiolaity

        I need to be able to take input from the enviroment to control my engine and logic.

      • Output Functiolaity

        I need to be able to give output form my engine and logic to the enviroment.

      • Execute Functiolaity

        I need to be able to execute engine and application logic.

      • Initiate Functiolaity

        I need to be able to initiate resources before the first iteration of the loop.

      • CleanUp Functiolaity

        I need to be able to free resources after the last iteration of the loop.
